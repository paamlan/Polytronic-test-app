# Privacy Policy
**Application : Polytronic 4.0 – Integration App**  
**Dernière mise à jour : 16 novembre 2025**

This Privacy Policy explains how Polytronic 4.0 (“the Application”) collects, stores, and protects user data.

## 1. Information We Collect
The Application may access the following data from QuickBooks Online:
- customers
- suppliers
- products/services
- invoices, receipts
- inventory information

No sensitive personal data is collected.

## 2. How Data Is Used
The Application uses data exclusively to:
- synchronize accounting and inventory records
- automate reporting
- generate dashboards and KPIs
- improve internal business processes

Data is never sold, shared, or used for advertising.

## 3. Data Storage and Security
Data is stored securely in encrypted environments (MySQL / secure cloud storage).  
Access is restricted to authorized Polytronic personnel.

## 4. Third-Party Services
The Application interacts only with:
- QuickBooks Online API
- Secure hosting environments (Azure, AWS, or equivalent)

No third parties receive user data.

## 5. User Rights
Users may request access or deletion of their data at any time by contacting the developer.
